## <kbd>Methods:</kbd>
**ChatView** has total 3 methods.

### 1. CreateChatView
Creates the chat view inside the given arrangement.

| Parameter | Type
| - | - |
| arrangement | component

### 2. AddMessage
Adds a message to the chat. If isSender is true, message is right-aligned.

| Parameter | Type
| - | - |
| message | text
| isSender | boolean

### 3. ClearMessages
Clears all chat messages.
